# Day 8 - LWC Basics

## What is LWC?

Lightning Web Components (LWC) is a modern JavaScript framework developed by Salesforce for building fast and reusable user interface components. It is based on web standards such as HTML, CSS, and JavaScript. LWC helps developers create responsive and efficient applications within the Salesforce platform.

---

## Why Salesforce Uses LWC

Salesforce uses LWC because it provides better performance, faster rendering, and improved developer experience compared to older frameworks. LWC follows standard web development practices, making it easier for developers to learn and build applications. It also improves component reusability, security, and maintainability in Salesforce applications.

## Component Breakdown

### 1. HTML File

Defines the structure of the component UI.

### 2. JavaScript File

Handles component logic, events, and data processing.

### 3. CSS File

Provides styling and improves UI appearance.

### 4. XML Configuration File

Makes the component available in Salesforce App Builder and other pages.

---

## Frontend vs Backend Logic

### Frontend Logic

* User interface design
* Handling button clicks and events
* Displaying data dynamically
* Form validation

### Backend Logic

* Fetching data from Salesforce database
* Business logic processing
* Apex methods integration
* Server-side operations
