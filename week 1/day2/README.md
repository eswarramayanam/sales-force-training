
 day-2 Platform basics 

 What is Salesforce Platform? 
- Salesforce platform is an App Development platform that extends your Customer Relationship Management systems reach and functionality.
- It has so many out of the box features and so much functionality.
- It provides Low-code Automation tools for development of apps. 
- Lightning UI for building apps.

 Platform basics
App
- It is a set of objects ,fields and other functionalities that supports a Business process.

Objects
- Objects are Tables in the Salesforce Database that store a particular kind of information.An Object is like a database table that stores information and every object contains Records and fields.

Tab
A Tab is a button or menu item used to open an object, app, report in a specific app in the playground.

 Difference: Configuration vs Coding
- Configuration
Configuration means customizing Salesforce apps without writing code. 
It is done using clicks and No programming languages needed.
Uses built in salesforce features to build the app.someone without any coding knowledge at all can build entire apps in Salesforce using prebuilt components and low-code tools and automation .

- Coding 
Complex logics and structures are built using coding tools .This is more flexible but Complex .
  - coding done using  :
     1. Lightning web component framework ( Java Script UI development Framework similar  to Angular JS or React
     2. APEX (uses Java Like Syntax)
     3. Node.js
